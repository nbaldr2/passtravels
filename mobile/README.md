# PassTravels Mobile

React Native (Expo) mobile app for travel visa intelligence.

## Setup

```bash
npm install
npx expo start
```

## Run

- `npx expo start` - Start Metro bundler
- `npx expo run:ios` - Run on iOS simulator
- `npx expo run:android` - Run on Android emulator

## Build

```bash
eas build --platform ios
eas build --platform android
```

## Features

- Passport rankings
- Visa requirements checker
- AI travel planner
- Interactive map
- User authentication
- Over-the-air updates with Expo Updates

## Updates

The app uses Expo Updates for over-the-air updates. For more information, see [docs/UPDATES.md](docs/UPDATES.md).
